## üó®Ô∏è Description

A **full-stack room-based real-time chat application** where users can create or join chat rooms and communicate instantly.

Frontend is built using **React**, backend using **Spring Boot**, and message communication is handled via **WebSocket (STOMP)**.  
All chat messages are stored and retrieved from **MongoDB** for persistence.

---

## üöÄ Features

- Multi-room chat system (create/join chat rooms)
- Real-time communication using WebSockets & STOMP
- Persistent storage of chat messages in MongoDB
- Modular architecture (React + Spring Boot decoupled)
- Scalable backend system with event-driven communication
